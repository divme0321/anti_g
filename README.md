## Monetization (Google AdSense)

Ad slots are pre-configured in the layout. To activate:

1. Sign up for [Google AdSense](https://adsense.google.com)
2. Get your ad unit code
3. Replace the placeholder in `src/main.js` (search for `ad-top` and `ad-bottom`)
4. Alternatively, use [Carbon Ads](https://www.carbonads.net/) for developer-focused ads

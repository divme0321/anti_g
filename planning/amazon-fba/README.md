## 📊 靴下OEMの競合分析

### 市場概要
| 指標 | データ |
|------|--------|
| 日本靴下市場規模（2025年） | 約1,904.7百万ドル（USD） |
| CAGR（2026-2033年） | 4.4% |
| 主要プレイヤー | Tabio、靴下屋、RoToTo、Nishiguchi Kutsushita |
| Amazon Japan内の競合度 | **非常に高い** ⚠️ |

### 靴下が高競合な理由
1. **大手の参入**: Tabio、無印良品、ユニクロなどが価格面で圧倒
2. **中国セラーの大量出品**: 低価格帯は中国セラーが飽和
3. **差別化が困難**: 機能素材（吸湿速乾等）は既に一般化
4. **レビュー数の壁**: 上位商品はレビュー1,000件以上で新規参入が困難

### 靴下で勝つなら（差別化戦略）
もし靴下に挑戦するなら、以下のニッチが有望：
- **5本指 × 消臭 × アーチサポート**の複合機能
- **登山・トレイルランニング特化**の超厚手靴下
- **和柄デザイン × 越境EC向け**（海外向け販売）
- **医療用着圧ソックス**（薬事法に注意）

## 📋 商品選定チェックリスト

各商品を選定する際は以下を必ず確認：

- [ ] Amazon BSR（ベストセラーランク）が10,000位以内か
- [ ] 上位3商品のレビュー数が500件以下か（参入余地）
- [ ] 販売価格1,500円〜5,000円の範囲か（FBA手数料を考慮）
- [ ] 利益率30%以上確保できるか（FBA手数料+送料込み）
- [ ] 季節変動が少ないか（Googleトレンドで確認）
- [ ] OEM可能な工場がAlibaba で3社以上あるか
- [ ] 小ロット（100〜300個）で発注可能か
- [ ] サイズ・重量がFBA標準サイズ内か
- [ ] 知的財産権の問題がないか
- [ ] 薬事法・PL保険の対象外か（リスク回避）

## 📈 Amazon FBA 開始ステップ

1. **リサーチ（1-2週間）**: 上記30商品から3〜5商品に絞り込み
2. **サンプル発注（2-3週間）**: Alibaba で3〜5工場からサンプル取り寄せ
3. **品質チェック（1週間）**: サンプル比較 → 工場選定
4. **本発注（2-3週間）**: 100〜300個の小ロットで初回発注
5. **商品ページ作成（1週間）**: A+コンテンツ、写真撮影、SEOキーワード最適化
6. **FBA納品（1-2週間）**: Amazon倉庫へ発送
7. **広告開始**: スポンサープロダクト広告でローンチ
8. **レビュー獲得**: Amazon Vine プログラム活用

> [!WARNING]
> **2026年のFBA手数料は平均6%上昇予測**。利益計算は手数料増加を織り込んで行うこと。

> [!TIP]
> **最初は1商品に集中**。3商品に分散するより、1商品を5つ星レビュー50件まで育てるほうが成功確率が高い。
